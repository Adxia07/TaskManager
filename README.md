#  TaskManager

> Диспетчер процессов с управлением приоритетами, CPU Affinity и деревом процессов.


---

##  Описание

TaskManager - это кроссплатформенный инструмент для мониторинга и управления процессами операционной системы.
Приложение позволяет в реальном времени наблюдать за запущенными процессами, менять их приоритеты, привязывать к конкретным ядрам процессора, просматривать потоки, строить дерево иерархии и визуализировать загрузку системы.

---
##  Возможности

- Показывает полный список действующих процессов с PID, названием, приоритетом, количеством потоков и количеством памяти.
- Сортировка по PID, названию, количеству потоков и памяти
- Возможность изменять приоритет процесса
- Возможность выбирать количество ядер  
- Построение дерева процессов
- Вывод графика CPU в графическом виде и с процентами
- Возможность выбора интервала обновления(изначально 20 секунд)
